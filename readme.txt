A simple plugins that uses jQuery and SWFObject!

== Description ==

A simple plugins that uses jQuery and SWFObject that included with WordPress,
jQuery is used to analyse every html object element, if a object element is
verified as a Flash object it will automatically register that Flash object
with SWFObject.

== Installation ==

1. Upload the unzipped folder `swfobjectjquery` to your `plugins` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
